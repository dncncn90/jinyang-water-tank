# Water Tank Website - Design System

This design system uses a masculine, professional aesthetic appealing to Korean male construction, plumbing, and facility management professionals.

## 1. Typography
- **Primary Font**: Sans-serif, Geist or standard system sans.
- **Weights**: Bold (700) for titles, Medium (500) for CTAs, Regular (400) for body.
- **Characteristics**: Large font size for legibility on mobile.

## 2. Colors
- **Primary**: Deep Navy `#0c4a6e` (var--color-industrial-900). Represents trust and durability.
- **Secondary**: Blue `#0ea5e9` (var--color-industrial-500). Action color for buttons.
- **Accent**: Orange `#f97316` or Red `#ef4444`. Used sparingly for "Urgent", "Discount", or "Hot" tags to grab attention.
- **Background**: White `#ffffff` with light gray `#f5f5f5` sections. Dark mode should use deep grays.

## 3. Components
- **Buttons**:
  - Primary CTA: large pad, bold text, slight shadow, #0ea5e9 background, #ffffff text. Slightly rounded corners (e.g., 4px-8px).
  - Secondary CTA: Outline button, #0c4a6e border, #ffffff background.
- **Cards**:
  - Simple borders (`1px solid #e5e7eb`), subtle shadows, clear padding. Top heavy with image, thick font title.
- **Tables/Forms**:
  - Striped rows for readability. Input fields with thick borders (`1.5px`) when focused.

## 6. Design System Notes for Stitch Generation

**DESIGN SYSTEM TO APPLY:**
- **Visual Style**: Clean, robust, and professional Korean B2B/B2C construction and plumbing supplies. 
- **Colors**: Use deep navy (`#0c4a6e`) for headers and primary trust elements. Use bright sky blue (`#0ea5e9`) for primary call-to-action buttons. Use white/light gray backgrounds to maximize contrast. Add sparingly orange (`#f97316`) for highlights or badges.
- **Typography**: Big, bold, legible sans-serif fonts. Headings should be prominent. Avoid thin or delicate fonts.
- **Layout**: Straightforward, blocky layouts with generous padding. Avoid unnecessary whitespace. Keep essential information "above the fold."
- **Components**: 
  - Buttons should be large and easy to tap (mobile-friendly), fully colored, with slightly rounded corners (`rounded-md`).
  - Cards should look sturdy, with a subtle border and shadow.
  - Forms use large input bars.
- **Vibe**: Trustworthy, fast, no-nonsense. "1분 만에 견적 받기" (Get a quote in 1 minute). Provide a strong sense of reliability.
